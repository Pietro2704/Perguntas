# Comandos Básicos de Git
## Inicio:

- Iniciar Repositório:
**git init**

- Guardar Mudanças:
**git add .**

- Confirmar Mudanças:
**git commit -m 'mensagem falando o que foi alterado'**

## Branches
- Listar branches:
**git branch**

- Criar nova_branch:
**git branch nova_branch**

- Trocar para a Branch Criada:
**git switch nova_branch**
